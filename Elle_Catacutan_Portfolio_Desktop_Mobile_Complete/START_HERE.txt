ELLE CATACUTAN PORTFOLIO — COMPLETE DESKTOP + MOBILE PACKAGE

This package contains the complete responsive website.

To publish:
1. Upload every file and the complete `assets` folder.
2. Keep `index.html`, `styles.css`, `script.js`, the CV and `assets` together.
3. Use `index.html` as the main page.

Do not upload `index.html` by itself.

Backup:
`Elle_Catacutan_Portfolio_Standalone_Backup.html` contains the website in one file and can be opened directly on a phone or computer.
