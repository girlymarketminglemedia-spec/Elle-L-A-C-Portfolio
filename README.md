## Publish on GitHub Pages
1. Unzip the folder.
2. Upload all files and the `assets` folder to the root of the GitHub repository.
3. In GitHub, open **Settings → Pages**.
4. Select the main branch and root folder.
5. Save and wait for the live link to update.
